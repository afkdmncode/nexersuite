API Documentation

Integrate AI tools into your applications

Authentication

Include your API key in the X-API-Key header or as a query parameter ?api_key=.

curl -X POST https://api.novaforge.ai/api/v1/chat \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your-api-key" \
  -d '{"messages": [{"role": "user", "content": "Hello"}]}'

Without a valid API key, requests use IP-based free credits.

Endpoints

POST /api/v1/chat

Send a chat message to the AI assistant.

Request: { "messages": [{"role": "user", "content": "..."}], "system": "..." }

Response: { "response": "...", "credits_remaining": 80 }

Credits: 5

POST /api/v1/ocr

Extract text from PDFs and images (multipart form).

Request: Multipart form with file field

Response: { "text": "extracted content..." }

Credits: 10

POST /api/v1/generate

Generate content (images, text) from prompts.

Request: { "prompt": "a cat in space" }

Response: { "response": "..." }

Credits: 20

POST /api/v1/translate

Translate text to another language.

Request: { "text": "Hello", "target_language": "Spanish" }

Response: { "translation": "Hola" }

Credits: 5

GET /api/v1/tools

List all available tools with their credit costs.

Response: Array of tool objects with slug, name, category, credit_cost
